# Rubrik Configuration Backup

## Overview

This tool provide backup and restore of the following Rubrik cluster configuration items:

* SLA Domains
* Archival Locations
* Replication Targets
* Fileset Templates
* Custom Report templates


## Pre-Requisites

* Python 3.7

## Usage

`rbkcb [-h] [--insecure] {backup,restore,status} path`
